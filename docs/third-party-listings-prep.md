## 为什么做这个

> Yext 研究（6,800,000 citations）：ChatGPT **48.7%** 引用来自第三方目录/评价平台。
> 你的网站内容再好，ChatGPT 也倾向于引用它信任的第三方平台来验证和推荐产品。
> **但关键是选对品类匹配的平台**——G2/Capterra 偏软件下载，不适合 KOC Engine。

## 1. Trustpilot（最重要，现在就做）

**网址：** https://business.trustpilot.com/

Trustpilot 是面向所有在线业务的评价平台，不像 G2 那样限定软件品类。ChatGPT 对 Trustpilot 的引用权重和 G2 同级。

### 注册步骤
1. 打开 https://business.trustpilot.com/ → "Get started"
2. 用公司邮箱注册
3. 搜索 "kocengine.com" → 如果没有自动识别，手动 claim 你的域名
4. 完善公司档案

### 公司档案信息（直接复制）

**Company name:**
```
KOC Engine
```

**Website:**
```
https://kocengine.com
```

**Category:**
```
E-commerce Service
Marketing Agency
```

**Company description（Trustpilot 限制较短）：**
```
KOC Engine connects cross-border e-commerce brands with vetted content creators. Brands post product collaboration tasks with built-in escrow protection. Creators browse the task hall, accept deals, and earn commission in platform points (1pt = $1 USD). AI matching finds the right creator in seconds. Dual-pledge system ensures both sides deliver.
```

### 邀请评价
- Trustpilot 注册后会给你一个 review invitation link
- 发送给早期用户，目标 **3 个月内 20+ 评价**
- Trustpilot 不需要 10 条才显示星级（比 G2 门槛低）

## 2. Influencer Marketing Hub 目录

**网址：** https://influencermarketinghub.com/

这是 influencer marketing 行业最大的媒体/目录站，品类 100% 匹配。

### 可以做几件事

**a) 提交到他们的工具目录**
- 找 "Submit Tool" 或 "List Your Platform" 入口
- 如果没有公开提交入口，直接联系 editorial@influencermarketinghub.com

**b) 争取被列入他们的 "Best Platforms" 文章**
- 当前已有文章如 "Best Influencer Marketing Platforms with Advanced Analytics"
- 联系编辑，提供你的差异化点（pledge escrow + cross-border focus + AI matching）

**c) 投稿 Guest Post**
- 基于已有的 Guide 页内容扩展
- 主题建议："Why Cross-Border Brands Need a Different Influencer Platform"

## 3. Crunchbase 公司档案

**网址：** https://www.crunchbase.com/

Crunchbase 是 LLM 做实体识别的基础数据源之一。免费创建，5 分钟搞定。

### 注册步骤
1. 打开 https://www.crunchbase.com/ → Sign up
2. 搜索 "KOC Engine" 确认没被占用
3. "Add Organization" → 选 "For-profit" → 填入信息：

**Organization name:** KOC Engine
**Website:** https://kocengine.com
**Founded:** 2026
**HQ:** [填实际城市或 Remote]
**Description:**
```
AI-powered bilateral matching platform for cross-border e-commerce KOC (Key Opinion Consumer) marketing. Connects Amazon sellers and DTC brands with vetted TikTok creators. Features dual-pledge escrow, AI smart matching, task hall, and real-time performance analytics. Commission in platform points (1pt = $1 USD).
```
**Categories:** E-Commerce, Marketplace, Influencer Marketing, Creator Economy
**Social:** Twitter, LinkedIn, GitHub（填已有的）

## 4. Google Business Profile（品牌搜索必备）

**网址：** https://business.google.com/

这不直接影响 LLM 引用，但：
- 品牌名搜索 "KOC Engine" 时，右侧会展示知识面板
- 和 Organization schema 互相验证，强化实体识别
- 免费的，5 分钟搞定

### 注册
1. 打开 https://business.google.com/ → "Manage now"
2. 搜索你的 business name，创建新档案
3. 填入网站地址、类别（选 "E-commerce service" 或 "Marketing consultant"）
4. **验证：** Google 会邮寄一张明信片到你的地址（填能收到邮件的真实地址）
